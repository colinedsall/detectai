# AI Text Detection System

A machine learning-based system for detecting AI-generated text using web-scraped human content and generated AI samples.

## Features

- **Web Scraping**: Collects long-form human articles from RSS feeds (BBC, NPR, Reuters, etc.)
- **AI Generation**: Creates diverse AI text samples using various topics and styles
- **ML Training**: Trains ensemble models (Random Forest + Logistic Regression) with probability calibration
- **Dynamic Confidence**: Intelligent confidence estimation based on probability, text length, and linguistic features
- **Direct Analysis**: Analyzes your own text files using the trained model
- **API Endpoint**: REST API for integration into applications
- **Configurable**: YAML-based configuration for easy customization
